What are arrays used for?
Arrays are used to store several pieces of information (of the same datatype) in a single variable.

What is the code for an int array?
int [] cartersnumbers = {1,2,3};
int [] shensnumbers = [3];

What is the differnce between Carter's array and Shen's array?
The first defines ints in an array, while the second defines the length of an int array.
